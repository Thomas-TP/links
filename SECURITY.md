# Security Policy

## Ownership

This repository is owned and maintained exclusively by Thomas Prud'homme (`@Thomas-TP`).

## Supported Versions

Security fixes are handled on the default branch only.

| Version | Supported |
| --- | --- |
| `main` | Yes |

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

If you discover a vulnerability, contact Thomas Prud'homme privately through the contact details listed on the GitHub profile or portfolio. Include:

- A clear description of the issue
- Steps to reproduce the problem
- The affected files, routes, or features
- Any proof of concept that helps verify the report

Reports will be reviewed as soon as possible. Confirmed issues will be fixed privately first, then disclosed only when appropriate.
